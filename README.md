# Tarea Unidad 4

Este proyecto es una aplicacion de consola que permite gestionar contenido multimedia como películas, series de televisión y documentales.

## Características

> - Agregar contenido multimedia (películas, series de televisión y documentales).
> - Eliminar contenido multimedia.
> - Mostrar contenido multimedia.
> - Guardar contenido multimedia en archivos CSV.
> - Cargar contenido multimedia desde archivos CSV.
> - Agregar elementos de producción (actores, temporadas, investigadores) a los contenidos multimedia.
> - Implementa Modelo Vista Controlador (MVC).

## Requisitos

- Java 17 o superior
- Maven 3.6 o superior
- Git 2.25 o superior

## Como clonar el repositorio

```bash
$ git clone https://github.com/Joniel-Ortiz/TareaUnidad4.git
```

## Como ejecutar el proyecto

```bash
$ mvn clean install
$ mvn exec:java -Dexec.mainClass="clases.Main"
```
